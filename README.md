# 📱 Exercício Celular

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de **programação orientada a objetos (POO)** através da criação de uma classe que representa um **celular** e suas funcionalidades básicas.

O projeto faz parte de exercícios acadêmicos voltados ao aprendizado de programação, permitindo aplicar conceitos importantes como **classes, atributos, métodos e organização de código**.

---

# 📌 Sobre o Projeto

O **Exercício Celular** consiste na implementação de uma estrutura de código que simula características de um celular, utilizando conceitos básicos de programação.

Durante o desenvolvimento foram praticados conceitos como:

* Criação de classes
* Definição de atributos
* Implementação de métodos
* Organização de código
* Aplicação de lógica de programação

Esse tipo de exercício é comum durante o aprendizado de programação, pois ajuda a desenvolver raciocínio lógico e compreensão de como estruturar programas.

---

# 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando:

* **Java**
* **Programação Orientada a Objetos (POO)**

# 📂 Estrutura do Projeto

A estrutura do projeto segue um modelo simples para facilitar o entendimento do código.

```bash
Exercicio_Celular
│
├── src/
│   └── classes relacionadas ao celular
│
└── main.java
```

A organização em arquivos separados facilita a manutenção e evolução do projeto.
